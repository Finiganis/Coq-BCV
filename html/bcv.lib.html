Dicoet Dicomore formalisent les maps (dictionnaires) ayant pour clés des nat et polymorphes sur les valeurs.

Il serviront pour représenter la mémoire, les programmes etc.

Module Dico:=Make(Nat_as_OT).

Tout le reste (définition supplémentaire et très nombreuses propriétés) est dans Dicomore.
Module Dicomore:=MoreMaps(Dico).

Une notation plus compacte pour find.
Notation FIND := (@Dico.find _).

Ce qui suit est spécifique à Dico car utilise le fait que les clés sont des entiers.

Génération de clés fraîches.

Fonction auxiliaire: retourne le plus grand index utilisé dans hp. Permet de retourner un index "frais" dans la suite.
Definition maxkey {A} (hp:(Dico.t A)) :=
  Dico.fold
    (fun k e acc ⇒ if (Compare_dec.lt_dec acc k) then k else acc) hp 0.

Propriété basique de maxkey: l'index retourné est bien "frais".
Lemma maxkey_correct : ∀ A (h:Dico.t A) k, k>(maxkey h) →
  ¬Dico.mem k h = true.
